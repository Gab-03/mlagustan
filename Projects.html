Infodemiology COVID-19 Monitoring Application

The Infodemiology COVID-19 Monitoring Application is a web-based platform designed to track and predict COVID-19 outbreaks across the Philippines. This application integrates Google Trends data and ARIMAX models to forecast case surges, providing real-time data visualization through interactive dashboards.

The project leverages syndromic surveillance methods, analyzing health-related search behaviors to predict potential outbreaks and assess public sentiment towards the pandemic. With a spatio-temporal approach, the platform generates risk maps and provides recommendations for resource allocation to hospitals, helping mitigate healthcare strain during critical times.

Additionally, the application incorporates health capacity data, offering real-time information on hospital availability and intensive care unit (ICU) capacity. The ultimate goal of this project is to enhance preparedness and response efforts for public health agencies in the Philippines during health crises.
